=============
gentoo-syntax
=============

A vim plugin that handles Gentoo and portage related syntax highlighting,
filetype, and indent settings.

Installing
==========

* Gentoo users: ``emerge app-vim/gentoo-syntax``
* Everyone else: ``make PREFIX=~/.vim/ install``

====
